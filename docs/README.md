# Avatar Generator — Documentation Site

The documentation site for [avatar-generator](https://github.com/hugoogb/avatar-generator), built with [Astro](https://astro.build) + [Starlight](https://starlight.astro.build).

## Commands

All commands are run from the `docs/` directory:

| Command        | Action                                     |
| :------------- | :----------------------------------------- |
| `pnpm install` | Install dependencies                       |
| `pnpm dev`     | Start local dev server at `localhost:4321` |
| `pnpm build`   | Build production site to `./dist/`         |
| `pnpm preview` | Preview production build locally           |

## Tech Stack

- **Astro** — static site generator
- **Starlight** — documentation theme
- **MDX** — interactive documentation pages with embedded components
