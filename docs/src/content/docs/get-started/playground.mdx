## Run Locally

You can also run the playground locally with the full package:

<Steps>

1. Clone the project

   <Code
     code={`git clone https://github.com/hugoogb/avatar-generator.git`}
     lang="sh"
   ></Code>

2. Install dependencies

   <Code code={`cd avatar-generator && pnpm install`} lang="sh"></Code>

3. Run the playground

   <Tabs syncKey="library">
     <TabItem id="core" label="core">
       <Code code={`pnpm playground:core`} lang="sh"></Code>
     </TabItem>
     <TabItem id="react" label="react">
       <Code code={`pnpm playground:react`} lang="sh"></Code>
     </TabItem>
   </Tabs>

4. Open your browser to the URL shown in the terminal

</Steps>

## Available Styles

| Style | Package | Description |
|-------|---------|-------------|
| Initials | `@avatar-generator/style-initials` | Letter initials on colored backgrounds |
| Geometric | `@avatar-generator/style-geometric` | Grid of shapes (circles, squares, triangles, diamonds) |
| Pixels | `@avatar-generator/style-pixels` | Symmetric pixel art patterns |
| Rings | `@avatar-generator/style-rings` | Concentric colored rings |

Each style produces unique, deterministic avatars based on the seed you provide.
