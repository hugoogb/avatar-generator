## Quick Start

Install the core package and at least one style:

<Tabs syncKey="package-manager">
  <TabItem id="npm" label="npm">
    <Code code={`npm install @avatar-generator/core @avatar-generator/style-initials`} lang="sh"></Code>
  </TabItem>
  <TabItem id="pnpm" label="pnpm">
    <Code code={`pnpm add @avatar-generator/core @avatar-generator/style-initials`} lang="sh"></Code>
  </TabItem>
  <TabItem id="yarn" label="yarn">
    <Code code={`yarn add @avatar-generator/core @avatar-generator/style-initials`} lang="sh"></Code>
  </TabItem>
</Tabs>

## Available Styles

Each style is a separate tree-shakeable package:

| Package | Description |
|---------|-------------|
| `@avatar-generator/style-initials` | Letter initials on colored backgrounds |
| `@avatar-generator/style-geometric` | GitHub-style symmetrical identicon patterns |
| `@avatar-generator/style-pixels` | 8-bit retro pixel faces |
| `@avatar-generator/style-rings` | Concentric rings with segmented/dashed variants |
| `@avatar-generator/style-faces` | Minimal flat geometric faces |
| `@avatar-generator/style-illustrated` | Detailed cartoon faces with accessories |
| `@avatar-generator/style-anime` | Anime/manga-inspired faces with expressive eyes |

Install any styles you want to use:

<Tabs syncKey="package-manager">
  <TabItem id="npm" label="npm">
    <Code code={`npm install @avatar-generator/style-geometric @avatar-generator/style-pixels @avatar-generator/style-rings @avatar-generator/style-faces @avatar-generator/style-illustrated @avatar-generator/style-anime`} lang="sh"></Code>
  </TabItem>
  <TabItem id="pnpm" label="pnpm">
    <Code code={`pnpm add @avatar-generator/style-geometric @avatar-generator/style-pixels @avatar-generator/style-rings @avatar-generator/style-faces @avatar-generator/style-illustrated @avatar-generator/style-anime`} lang="sh"></Code>
  </TabItem>
  <TabItem id="yarn" label="yarn">
